# Scripts

## Propósito

Armazenar scripts de geração e utilitários de desenvolvimento para o frontend.

## O que faz

- Contém ferramentas para gerar entidades e converter nomes de arquivos/campos.
- Auxilia no mapeamento de estruturas de dados entre backend e frontend.
- Não faz parte do runtime da interface; é usado apenas em build ou durante o desenvolvimento.

## Diretrizes

- Mantenha scripts focados em tarefas de geração e transformação de arquivo.
- Não implemente lógica de UI ou componentes aqui.
- Use `README-entity-generator.md` como referência para execução e convenção.
- Atualize a documentação do script sempre que mudar a forma como as entidades são geradas.
